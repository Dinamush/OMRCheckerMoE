# `.shared/` — optional local tooling

This directory is for **machine-local** bundles that should not be committed.

## ui-ux-pro-max

Install the Cursor **ui-ux-pro-max** skill here so paths match the command:

`python .shared/ui-ux-pro-max/scripts/search.py …`

1. Clone or copy [`nextlevelbuilder/ui-ux-pro-max-skill`](https://github.com/nextlevelbuilder/ui-ux-pro-max-skill).
2. Put the `src/ui-ux-pro-max` contents into **`ui-ux-pro-max/`** next to this file (full path: `.shared/ui-ux-pro-max/`).

After install, see **`ui-ux-pro-max/README.md`** inside that folder for example commands. The bundle is listed in the repo root `.gitignore`.
