# Archive

These scripts are **not used** by the main web application (`main.py` in the project root). They are kept for reference or standalone use only.

- **ph.py** – Standalone PornHub scraper
- **xh.py** – Standalone xHamster scraper (CLI, manual login)
- **pixiv.py** – Pixiv-related script
- **youtube.py** – YouTube-related script
- **video_scraper.py** – Generic video scraper
- **telegram.py** – Telegram-related script
- **UI_version/** – Alternate UI implementation (main.py, downloader.py)

The active system is the **Unified Video Downloader** run via `main.py` at the project root. See the main [README.md](../README.md) for usage.
