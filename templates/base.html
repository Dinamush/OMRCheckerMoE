Safety & responsibility

  • Use this app only for content you are allowed to download and store. Respect each site's terms of service and copyright.
  • This tool automates a browser on your PC; credentials and sessions stay local to your machine.
  • Deleting files in the duplicate checker is permanent. Confirm paths before you confirm deletion.

Tools

  • Video downloader — favourites-based downloads (manual login in browser).
  • Duplicate file checker — scan any folder, preview videos, remove extras.
  • Library browser — thumbnail grid of all downloaded videos.
  • Integrity scanner — detect truncated or corrupt video files.
  • Session history — log of all completed download runs.

Running the server

  • Start with python main.py from the project folder (default UI port is shown in the terminal).
  • Default downloads/logs live under your OS user profile (e.g. Windows %LOCALAPPDATA%\HamsterScraper), not next to the app install.
  • For issues, check application logs and the download folder you configured.
